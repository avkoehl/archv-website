ARCHV

Archive vision provides a model for the computational analysis of large sets of images. Foremost, it is a tool for pattern recognition, finding recurring visual elements within the set and quantitatively ranking their differences. The application uses SURF feature detection and extraction to find keypoints within the images to be compared. The keypoints are then matched using FLANN and then filtered based on their homography.
