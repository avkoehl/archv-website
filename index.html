Scan Database

Archv (Archive-vision) is a tool to search an imageset to find the best matches with your seed image. This page allows you to upload and compare an image of your choice with either the English Broadside Ballad Association imageset or a curated image set from the British Library's Flickr.

Go to the overview to learn how it works.


First

Select the Image set that you would like to compare your image to:

Ebba BL-flickr


Second

Upload your image as the seed image. When the file is uploaded, the keypoints overlayed on the image will be displayed.

Third

Click the scan Database button to see the matches found within the image set. It will take about 15 to 30 seconds to complete.
