Demo Archv

This page offers a demonstration of how Archv works by demonstrating the main process using an example impression from EBBA. First, press the Show Keypoints button to see the keypoints that are found on the image. Then, press the Scan Database button to see the top matching images for this example Please note, it will take about thirty seconds to scan the database for matches once the button is pressed
